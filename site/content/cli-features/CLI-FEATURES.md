# CLI Features

Возможности командной строки Claude Code, которые **не являются файлами конфигурации**. Это встроенные фичи терминала, управляемые через слэш-команды и настройки.

## Обзор

| Фича | Команда | Назначение |
|---|---|---|
| **Fast Mode** | `/fast` | Ускорение Opus 4.6 в 2.5 раза |
| **Checkpointing** | `/rewind` | Откат изменений к предыдущему состоянию |
| **Sandboxing** | `/sandbox` | Изоляция файловой системы и сети |

## Отличие от конфигов

Эти фичи:
- Не создают файлы в `.claude/` или `~/.claude/`
- Управляются через CLI команды или `settings.json`
- Являются возможностями runtime, а не структурой проекта

```
# Это файлы конфигурации
.claude/settings.json
~/.claude/keybindings.json

# Это CLI фичи (не файлы)
/fast      → переключатель режима
/rewind    → навигация по checkpoint'ам
/sandbox   → выбор режима изоляции
```

## Где настраиваются

Хотя сами фичи не являются файлами, их поведение можно настроить через `settings.json`:

```json
{
  "fastMode": true,
  "fastModePerSessionOptIn": true,
  "sandbox": {
    "enabled": true,
    "filesystem": {
      "allowWrite": ["~/.kube"]
    }
  }
}
```

## Связь с терминалом

В интерактивном терминале этого проекта (`/help`) вы можете увидеть эти команды в действии:

- `/fast` — переключение fast mode
- `/rewind` — открытие меню отката (в реальном Claude Code)
- `/sandbox` — выбор режима песочницы (в реальном Claude Code)
